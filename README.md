# Course Allotment Optimization System

## Problem
Manual course allocation is inefficient and may not satisfy professor preferences.

## Solution
This project models course allocation as a weighted bipartite matching problem 
to maximize total satisfaction score.

## Features
- Graph-based modeling
- Weighted preference optimization
- Satisfaction percentage calculation
- Modular C++ project structure

## Tech Stack
- C++
- Graph Algorithms
- Optimization Techniques
